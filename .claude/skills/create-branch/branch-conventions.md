# Branch Conventions

## Branch Prefix

| prefix | 説明 |
| --- | --- |
| feature | 新機能 |
| fix | バグ修正 |
| refactor | リファクタリング |
| docs | ドキュメント |
| chore | 雑務・メンテナンス |

## Branch Name Format

| ルール | 説明 |
| --- | --- |
| 形式 | `prefix/descriptive-branch-name` |
| 区切り文字 | ハイフン |
| 文字数制限 | prefixを含むブランチ名で40文字以内 |
| 対象の明示 | モジュール、サービス、ツール、ファイル名を含める |

### Examples

`feature/user-auth-api`
