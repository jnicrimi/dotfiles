# PR本文テンプレート

## 出力ルール

### 構成

- テンプレートのセクション構成（見出し）は変更しない
- `[...]` のプレースホルダは差分から適切な内容を生成して置き換える
- 本文に既存の内容がある場合は、生成した本文の末尾にそのまま残す

### 記載内容

- 差分から読み取れない情報（背景・意図・運用方針など）は本文に含めない
- 具体的なファイル名やコードは含めない
- 変更点は重要度の高い上位 5 件まで表示する（変更箇所が少ない場合は必要最低限のみ）

### 文体・表記

- 箇条書きの文体は **体言止め（〜追加／〜変更／〜改善／〜有効化 など）** で統一する
- 箇条書きの行末に句点をつけない
- 日本語と半角英数字・インラインコードの間に半角スペースを入れる
- マークダウンの強調表示（`**`, `*`, `__`）は使用しない

## テンプレート

```markdown
## リリースノート

- [変更点]
- [変更点]
- [変更点]
- [変更点]
- [変更点]
```
